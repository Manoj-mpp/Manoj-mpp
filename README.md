### Smart Irrigation System ‚Äî Team Project
*Nov 2025*
*Tech Stack: ESP8266, DHT11, Soil Moisture Sensor, LDR, I2C LCD, IoT*
- Developed a low-cost smart irrigation system using **ESP8266** to monitor temperature, humidity, and soil moisture in real time.
- Integrated **DHT11**, soil moisture sensor, and **LDR** with a **16x2 I2C LCD** for continuous live environmental data display.
- Implemented automatic pump control using a relay module, activating irrigation when soil moisture drops below threshold.
- Added **LDR-based daylight sensing** to ensure pump operation only during daytime, improving efficiency and reducing energy use.
- Designed an **IoT-ready architecture** enabling future extension to cloud monitoring and mobile app-based control.
